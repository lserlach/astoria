# Астория — прототип (Next.js)

Кликабельный фронтенд без бэкенда: мок-данные и имитация задержки сети.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000). Если порт занят, Next.js предложит другой (см. вывод в терминале).

## Маршруты

| Путь     | Описание                          |
| -------- | --------------------------------- |
| `/`      | Главная, переходы в поиск         |
| `/search`| Поиск туров и отелей (мок-фильтры)|

Параметры запроса (опционально): `?to=Турция`, `?from=Москва` — подставляются в форму поиска.

## Моки и API-слой

- Типы и фасад: [`lib/api/index.ts`](lib/api/index.ts), [`lib/api/types/index.ts`](lib/api/types/index.ts)
- Реализации с `delay()`: [`lib/api-mock/`](lib/api-mock/)

Для подключения бэкенда замените реализацию в фасаде `lib/api/index.ts` (или отдельном адаптере), страницы менять не обязательно.
